配置ssh连接

#####windows

以下命令都是在Git Bash中执行的

  • 生成密钥对（SSH key）

    ssh-keygen -t rsa -b 4096 -C "xxx@xxx.com"

    其中，xxx@xxx.com 是邮箱，就是一个标识信息，它会写到公钥的最后面

  • 将 SSH key 添加到 ssh-agent

  • 将 SSH key 添加到 Github 账户

    打开 ~/.ssh/id_rsa.pub文件,拷贝全部内容，或者输入命令拷贝到剪贴板

    clip < ~/.ssh/id_rsa.pub

    然后登录git账号，将SSH Key加入你的SSH key中

  • 测试 SSH 连接

    ssh -T git@github.com
